Tag <html/>

Definição

A tag <html> representa a raiz de um documento HTML.

A tag <html> é uma tag que funciona como conteiner para todos os outros elementos HTML (exceto para a tag <!DOCTYPE>)

Nota: Você deve sempre incluir o atributo lang dentro da tag <html> para declarar o idioma da página da Web. Isto destina-se a ajudar os motores de busca e navegadores.

Atributos

Atributo Valor Descrição
xmlns link Especifica o atributo de namespace XML (se você precisar que seu conteúdo esteja em conformidade com XHTML).

Essa tag também suporta os atributos globais.

Exemplos

                    
                        <!DOCTYPE html>

                        <html lang="en">
                        <head>
                        <title>Título do documento</title>
                        </head>
                        <body>
                        <h1>Esse é um cabeçalho!</h1>
                        </body>
