Tag <ol/>

A tag <ol> define uma lista ordenada. Uma lista pode ser ordenada alfabeticamente ou numericamente.

A tag <li> define cada item da lista.

Dica: Use CSS para estilizar a lista.

Dica: Para lista não ordenadas use a tag <ul>.

Atributos

Atributo Valor Descrição
reversed reversed Especifica que a ordem da lista deveria ser invertida.
start numero Especifica o valor inicial da lista.
type 1
A
a
I
i		 Especifica o tipo de marcador usado na lista.

Essa tag também suporta os atributos globais e de eventos.

Exemplos

            
            <ol style="list-style-type:upper-roman">
                <li>Coffee<</li>
                <li>Tea</li>
                <li>Milk</li>
            </ol>

            <ol style="list-style-type:lower-alpha">
                <li>Coffee</li>
                <li>Tea</li>
                <li>Milk</li>
            </ol>
