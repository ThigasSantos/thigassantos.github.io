Tag <colgroup/>

Definição

A <colgroup> tag especifica um grupo de uma ou mais colunas em uma tabela para formatação.

A <colgroup> tag é útil para aplicar estilos a colunas inteiras, em vez de repetir os estilos para cada célula, para cada linha.

Observação: a <colgroup> tag deve ser filha de um elemento <table>, após qualquer elemento <caption> e antes de qualquer elemento <thead>, <tbody>, <tfoot> e <tr>.

Dica: Para definir diferentes propriedades para uma coluna dentro de um <colgroup>, use a tag <col> dentro da <colgroup>tag.

Atributos

Atributo Valor Descrição
span number Especifica o número de colunas que um grupo de colunas deve abranger.

Essa tag também suporta os atributos globais e de eventos.

Exemplos

            
                <table>
                    <colgroup>
                        <col span="2" style="background-color:red">
                        <col style="background-color:yellow">
                    </colgroup>
                    <tr>
                        <th>ISBN</th>
                        <th>Title</th>
                        <th>Price</th>
                    </tr>
                    <tr>
                        <td>3476896</td>
                        <td>My first HTML</td>
                        <td>$53</td>
                    </tr>
                </table>
