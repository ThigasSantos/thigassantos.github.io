A tag <ul> define uma lista ordenada.

A tag <li> define cada item da lista.

Dica: Use CSS para estilizar a lista.

Dica: Para lista não ordenadas use a tag <ol>.

Atributos

Essa tag suporta os atributos globais e de eventos.

Exemplos

            

                <ul style="list-style-type:circle">
                  <li>Coffee</li>
                  <li>Tea</li>
                  <li>Milk</li>
                </ul>
