Tag <source/>

Definição

A tag <source> é usado para especificar vários recursos de mídia para elementos de mídia, como <video>, <audio> e <picture>.

A tag <source> permite especificar arquivos alternativos de vídeo/áudio/imagem que o navegador pode escolher, com base no suporte do navegador ou na largura da janela de visualização. O navegador escolherá o primeiro <source> que ele suporta.

Atributos

Atributos

Atributo Valor Descrição
media media_query Aceita qualquer consulta de mídia válida que normalmente seria definida em um CSS.
sizes Especifica tamanhos de imagem para diferentes layouts de página.
src URL Necessário quando a tag é usada em <audio> e <video>. Especifica o URL do arquivo de mídia.
srcset URL Necessário quando a tag é usada em <picture>. Especifica a URL da imagem a ser usada em diferentes situações.
type MIME-type Especifica o tipo MIME do recurso.

Essa tag também suporta os atributos globais e de eventos.

Essa tag suporta os atributos globais e de eventos.

Exemplos

Use a tag <source> com <video> para executar um vídeo.

            
            <video width="320" height="240" controls>
                <source src="movie.mp4" type="video/mp4">    
            </video>
