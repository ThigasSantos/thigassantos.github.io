Tag <strike/>

Definição

Não é suportada no HTML5.

A tag <strike> era usada no HTML4 para definir o texto tachado.

Exemplos

No lugar da tag <strike>, podem ser utilizadas as tags <del> e <s>.

Use a tag <del> para definir o texto excluído:

                    
                      <p>
                        Minha cor favorita é <del>azul<del> <ins>vermelho<ins>.
                      </p>

Use a tag <s> para marcar o texto que não está mais correto:

                    
                      <p>
                        <s>Meu carro é azul.</s>
                      </p>
